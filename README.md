# Comprehensive Association Management System

The Comprehensive Association Management System is a secure, web-based platform designed to streamline and centralize the operations of residential associations. It provides features for association members to efficiently manage key functions such as:

* Resident Records
* Association Members
* Vendor Management
* Staff Coordination
* Facility Usage
* Meeting Records

## Features

* **Resident Management**
  Maintain owner/tenant records, parking, maintenance payments, and flat details.

* **Association Member Database**
  Store and manage member profiles, contact information, and association roles.

* **Vendor Management**
  Track vendor details, service history, work logs, costs, and automated entry/exit reminders.

* **Staff Management**
  Monitor staff check-in/check-out, work assignments, attendance, and salary tracking.

* **Facility Usage Tracking**
  Track usage of community facilities such as gyms and swimming pools with automated reports.

* **Meeting Records**
  Store previous meeting details, discussions, and allow attachment of minutes and documents.

## Data Structures Used

* Structure (struct)
* Array
* Singly Linked List
* Min Heap
* Max Heap
* Binary Search Tree
* File System (CSV)

## Project Structure

* **Frontend:**
  HTML, CSS, JavaScript (Web-based User Interface)

* **Backend:**
  C Programming (Compiled Executables)
  Node.js (Controller and Routing)

* **Data Storage:**
  CSV Files (Resident, Vendor, Staff, Facility, Meeting Records)
